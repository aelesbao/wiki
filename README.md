## How do I work with wiki?

### Preparation
- You need to globally install `Python: ^3.7.2`
- Run `make init` for install dependencies

### For start locally
- Run `make start`
- Go to http://127.0.0.1:8000

### For deploy changes in the wiki to web (only for contributors)
- Run `make deploy`
